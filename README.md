## Взаимодействие в WWW: взаимодействие с сервисами

**Этап 1. Подготовка данных**

Скачайте файл с набором данных событий информационной безопасности в формате JSON (events.json). Важно: можно сгенерировать данные самостоятельно или использовать предоставленные.

**Этап 2. Анализ данных.**

Загрузите данные из файла JSON в датафрейм с помощью Pandas
и проанализируйте их. Вы должны изучить распределение событий информационной безопасности по их типам (по полю «signature»).

**Этап 3. Визуализация данных.**

Постройте график распределения типов событий информационной
безопасности. Используйте Seaborn или Matplotlib для создания визуализации.
